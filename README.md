# Controle Financeiro

Sistema em Python desenvolvido para automação e gerenciamento de fluxo de caixa da loja Bananinha Materiais para Construção.

## 🚀 Funcionalidades
* Cadastro e controle de contas a pagar.
* Geração de relatórios de boletos pagos.
* Interface visual para facilidade de uso.

## 🛠️ Tecnologias Utilizadas
* Python 3.14.0
* SQLite (Banco de dados)
* Bibliotecas utilizadas: [Tkinter/Tkcalendar, OpenPyXL, Babel e PyInstaller] 

## 📦 Como Executar o Projeto

1. Clone o repositório:
   git clone https://github.com/ssouza509-lang/controleFinanceiro

2. Instale as dependências:
   pip install -r requirements.txt

3. Execute o sistema:
   python main.py
